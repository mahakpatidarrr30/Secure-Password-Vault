# Secure Password Vault 
A secure and user-friendly desktop application built with Java, JavaFX, CSS, HTML, and SQL for managing and storing passwords safely. The vault provides encrypted password storage, user authentication, and a clean, modern GUI for easy interaction.
## Features
### User Authentication
Login system to ensure only authorized users can access the vault
### Encryted Password Storage
All stored credentials are securely encrypted before being saved in the SQL database.
### Password Management (CRUD)
● Add new credentials
● View existing credentials
● Update saved entries
● Delete entries
